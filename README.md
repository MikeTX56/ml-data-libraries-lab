# ML Data Libraries Lab

A hands-on learning repository focused on mastering the core Python data science libraries:

- NumPy
- Pandas
- Matplotlib

The goal of this repository is to build strong foundations in numerical computing, data analysis, and data visualization through practical projects.

---

## Repository Structure

```text
ml-data-libraries-lab/
│
├── numpy-projects/
├── pandas-projects/
├── matplotlib-projects/
│
├── datasets/
├── utils/
│
├── requirements.txt
└── README.md
```

## Learning Goals

This repository focuses on developing intuition and practical skills in:

- Numerical computation with NumPy
- Data manipulation and analysis with Pandas
- Data visualization with Matplotlib
- Exploratory Data Analysis (EDA)
- Building reusable analysis tools

## Projects

Each folder contains independent projects designed to practice real-world data workflows.

### NumPy Projects
Focus on numerical computation and implementing algorithms from scratch.

Examples:
- Linear Regression from scratch
- K-Means clustering implementation
- Monte Carlo simulations
- Image processing with arrays

### Pandas Projects
Focus on real dataset exploration and data cleaning.

Examples:
- Netflix dataset exploration
- World population analysis
- Data cleaning pipelines

### Matplotlib Projects
Focus on visualizing data and understanding patterns.

Examples:
- Distribution visualization
- Correlation heatmaps
- Time-series plots
- Machine learning algorithm visualization


## Requirements

Install dependencies with: 
``` text
 pip install -r requirements.txt
```
Main libraries used:

- numpy
- pandas
- matplotlib
- jupyter

## Purpose

This repository documents the learning journey toward becoming a machine learning engineer by building strong foundations in data analysis and numerical computing.
